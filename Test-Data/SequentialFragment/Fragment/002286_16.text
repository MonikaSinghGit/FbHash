 not  require
extensive servicing, Appellant has met its burden of establishing
clear  error  in  the CO's SIC code designation.  Therefore,  the
appropriate SIC code for the procurement is 7359.


                         III. CONCLUSION

     The  Administrative Judge REVERSES the CO's  designation  of
SIC   code  5063,  Electrical  Apparatus  and  Equipment,  Wiring
Supplies, and Construction Materials, for this procurement.   The
Administrative Judge concludes instead the appropriate  SIC  code
is 7359, Equipment Rental and Leasing, Not Elsewhere Classified.
     
     This  is the Small Business Administration's final decision.
